# 关于

 `FlexTree`是一款基于`左右值`算法的树存储管理组件，适用于`Node.js`环境。提供了高效的树结构存储和访问，支持多种树操作，如增删改查、遍历、移动、查询等。

**功能特点：**

- 基于左右值算法，高效的树结构存储和访问
- 简单易用的API
- 强大易用的多种树操作，如增删改查、遍历、移动、查询等
- 采用`TypeScript`开发，提供完整友好的类型定义
- 支持任意数据库存储，如`SQLite`、`MySQL`、`PostgreSQL`等
- `95%`的测试覆盖率，保证代码质量
- 适用`Node.js`环境
